# KONGTONE

<div align="center"><i>NYU MS in Publishing Capstone Project</i></div>

KongTone is a Cantonese learning website. The naming had multiple layers: it sounds like 'Canton', consists Hong 'Kong', and means 'to make sense' in Cantonese. 

In KongTone, we aim to bring a gamified learning experience
